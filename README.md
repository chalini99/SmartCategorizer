# Smart-CategorizerAI (prototype)

## Quick start
1. install: pip install -r requirements.txt
2. train: python train_model.py
3. predict examples: python predict.py
4. evaluate: python evaluate.py
5. show explainability: python explain_top_features.py
6. feedback (interactive): python feedback_loop.py
7. retrain including feedback: python retrain_with_feedback.py

Artifacts (created): artifacts/model_pipeline.joblib, artifacts/metrics.json, artifacts/confusion_matrix.png
